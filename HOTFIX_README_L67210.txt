本包修复：
1. 桌面桥接每次发送都会启动新的后端子进程，本包新增跨子进程会话历史文件，避免第二轮开始丢上下文、重复模板回复。
2. 桌面端启动时自动切换桌面 Prompt，不再使用“最小 CLI 模式 / 不具备桌面端能力”的矛盾提示词。
3. 普通聊天在 planner_mode=rule_only 时直走模型对话，不再先走 Planner 执行链，不再外露“未生成可执行计划”。
4. stdout / stderr 分离：成功响应只读取 stdout；错误信息只在失败诊断中出现。
5. 清理规则支持中文逗号、英文逗号、全角分号、半角分号，并过滤 [计划器]/[运行链] 噪声。
6. 清屏/新会话会同时请求本地桥接清除桌面端会话历史文件，不删除 Runtime 审计、记忆或任务记录。
