五、新增状态对象作用与边界

Skill 状态对象：记录 Skill 可见、选择、激活、失败事实；不读取 Skill，不选择 Skill，不激活 Skill，不生成工具组。

ToolGroup 状态对象：记录工具组声明、可见、释放、租约事实；不创建工具，不展示工具，不释放工具，不续租工具，不持有可执行函数。

ToolIntent 状态对象：记录模型工具意图、参数摘要、工具组、Skill、运行任务和边界状态；不执行参数校验器，不做风险评分，不调用工具。

Model 状态对象：记录模型请求、响应、反馈、反思引用；不新增 ModelPort，不调用模型，不消费真实流式响应，不生产候选。

Action / Effect 状态对象：记录动作意图和效果观察引用；不执行动作，不采集观察，不读取屏幕、文件、网络或真实环境。
