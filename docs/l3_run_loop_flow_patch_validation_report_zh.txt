天工造物 L3 主运行编排 Flow 最小补丁 Validation 报告

一、验证结论

验证通过。新增 Flow 为被动规格，不是真实运行循环。

二、边界验证

1. Flow 类全部可从 tiangong_kernel.l3_orchestration 导入。
2. Flow 类均为 frozen + slots。
3. request_only/advisory_only/reference_only/no_execution/no_decision/no_persistence 默认均为 True。
4. 将上述布尔字段置 False 会抛 ValueError。
5. CanonicalRunLoopFlowBundle 包含主链节点。
6. EffectRequestFlow、HumanApprovalFlow、ScheduleTriggerTimerFlow、EventAppendFlow、AuditFlow、DecisionFlow、LeaseValidationFlow 均只表达引用，不执行、不写、不裁决、不签发。
7. L3ToL4/L5/L6 handoff 仅新增 flow_refs 引用字段。

三、验证命令

1. python -m compileall -q tiangong_kernel tests
2. python -m pytest -q tests -k "l3_run_loop_flow or l3_phase8 ..."
3. python -m pytest -q tests -k "l4_phase1 or l4_phase8 or l3_compat"

四、结果

1. compileall：通过。
2. L3 run loop + L3 phase8 分片：34 passed, 822 deselected。
3. L4/L3 兼容分片：49 passed, 807 deselected。
