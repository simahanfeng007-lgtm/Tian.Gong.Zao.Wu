二、论文/生物学依据转工程口径

1. 情绪不是固定标签，而是随时间变化的动态状态。
工程转化：AffectiveStateEngine 使用 temporary_delta、half_life、EMA、allostatic_load、affective_momentum 表示临时波动、衰减、压力负载和状态动量。

2. 情绪、动机与行动倾向在生物学和心理学中高度耦合，但不能简单等同。
工程转化：七情影响 language_state_logic；六欲影响 doing_mode_logic；二者可耦合计算，但最终权限边界严格分离。

3. 认知评价 + 强化学习模型强调 novelty、goal relevance、goal conduciveness、power 等评价信号。
工程转化：七情七源和六欲六源全部来自归一化信号源，不用硬编码情绪标签。

4. 稳态/异稳态调节适合建模资源压力、疲劳、恢复和边界稳定。
工程转化：六欲 survival/rest 与 allostatic_load 共同影响长链节奏、压缩建议、恢复耐心，但不能拒绝合法请求。

四、七情七种信号源

SevenEmotionSignalSources：
1. joy_reward_signal：喜，来自成功、收益、完成感。
2. obstruction_violation_signal：怒，来自阻滞、重复失败、边界破坏。
3. uncertainty_future_risk_signal：忧，来自不确定性、悬而未决、未来风险。
4. reflection_load_signal：思，来自复杂推理、复核负载、结构化需求。
5. loss_failure_signal：悲，来自失败、损失、不可恢复结果。
6. threat_irreversible_signal：恐，来自 A5、凭证、隐私、不可逆副作用。
7. novelty_prediction_error_signal：惊，来自新奇、异常、预测误差。

五、六欲六种信号源

SixDesireSignalSources：
1. survival_resource_boundary_signal：生存/边界稳态，来自资源压力、边界压力、不可逆风险。
2. curiosity_knowledge_gap_signal：好奇/学习探索，来自知识缺口、新奇信号。
3. achievement_goal_gap_signal：成就/闭环推进，来自目标差距、未完成任务。

6. rest_fatigue_recovery_signal：休整/恢复，来自疲劳、上下文压力、资源压力。

七、临时波动动态公式

核心组合规则：
current_total = clamp01(soul_baseline + temporary_delta)

临时波动计算：
retention = exp(-elapsed_seconds / half_life_seconds)
decayed_previous_delta = retention * previous_delta
target_delta = target_signal - soul_baseline
next_delta = EMA(decayed_previous_delta, target_delta, alpha)

关键参数：
- emotion_alpha=0.38
- desire_alpha=0.32
- emotion_half_life_seconds=1800
- desire_half_life_seconds=3600
- temporary_delta_limit=0.72
- stress_coupling_strength=0.12

含义：
- 临时情绪可大幅波动，但有上限。
- 无新刺激时会逐步回到 Soul 底色。
- Soul 改变后，旧临时波动默认重置。
- 压力、失败、威胁、疲劳会提高 allostatic_load，但不构成拒绝权。
