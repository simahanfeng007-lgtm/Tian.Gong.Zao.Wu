# 天工造物 L6.11 长链 Runner 与插件建议桥使用说明

## 定位

L6.11 在 L6.10 受治理真实运行链基础上，新增两项能力：

1. 长链 Runner：支持多步骤任务一次性进入治理链执行。
2. 插件建议桥：允许 L6 插件输出 PlanSuggestion / RepairSuggestion / QualityGateSuggestion / HandoffSuggestion，但建议不能直接执行，必须转换为 ToolInvocation 后进入 ExecutionSpine。

## 命令示例

```powershell
python run_agent.py --mock --tool-mode runtime_governed --workspace ./workspace --once "write demo.txt :: hello && read demo.txt && zip . dist/demo.zip"
```

执行链：

```text
Shell
→ RuntimeEntry
→ PlanBridge
→ LongChainRunner
→ ExecutionSpine
→ RiskClassifier
→ PermitGateway
→ RuntimeToolRegistry
→ Adapter
→ AuditBridge
→ PublicProjectionBridge
```

## 风险策略

- A1 只读：自动执行。
- A3 工作区写入 / 质量检查 / 打包：自动执行。
- A4：生成确认票据，暂停。
- A5：硬阻断。

## 插件建议桥规则

插件建议只能产生计划建议，不能直接执行。

允许建议工具：

- list_dir
- read_file
- write_workspace_file
- run_python_quality_check
- create_zip_package

禁止：

- terminal_shell
- 任意 shell
- 桌面控制
- 外部网络下载执行
- 直接跳过 PermitGateway

## 当前限制

- 未实现真实用户确认票据恢复执行。
- 未接真实 L6 插件运行时自动产出建议；当前是对象级桥接与测试验证。
- 未开放任意终端命令。
- 未做桌面端和安装包。
