L6.34 CLI 会话连续性验证报告

验证范围：
1. SessionState.build_context_hint 能压缩最近非 system 对话。
2. run_runtime_task 将 CLI 最近会话上下文注入 RuntimeEntry.run_text 的 external_context_hint。
3. preview_runtime_plan 将 CLI 最近会话上下文注入 RuntimeEntry.preview_plan。
4. run_planner_execution_task 支持 external_context_hint。
5. plan_failed fallback 仍保留最近两轮对话正文。
6. run_runtime_task 成功后继续写入 [运行链上下文摘要]。

验证用例：tests/test_l6_34_model_cli_continuity.py
结果：8 passed

关键行为：
- 用户说“上面代码 / 上一步结果”时，Planner user message 中会携带 CLI 最近会话摘要。
- 纯代码/纯分析/answer JSON 会进入 return_code / return_analysis 虚拟工具审计链。
- 危险 shell、绝对路径不会被虚拟返回兜底绕过。
- DeepSeek plan 样本回放 14/14 matched_expectations，accepted_expected_rate=1.0。

边界：
- L6.34 不做真实 Provider 网络调用。
- L6.34 不做正式 Skill 激活。
- L6.34 不写长期记忆，只做进程内安全摘要与 CLI 最近会话提示。
