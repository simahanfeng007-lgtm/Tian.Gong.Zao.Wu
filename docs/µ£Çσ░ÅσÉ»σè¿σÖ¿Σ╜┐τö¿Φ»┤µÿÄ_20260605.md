# 天工造物 L6.9 最小智能体启动器使用说明

## 定位

L6.9 启动器是外壳层，不是 L6 主体内核的一部分。它用于把已经通过测试的内核工程包装成最小可交互 CLI 智能体。

当前支持：

- `python run_agent.py --mock --once "你好"`
- `python run_agent.py --mock`
- `python -m tiangong_agent_shell --mock --once "你好"`
- OpenAI-compatible 非流式最小调用
- API Key 脱敏显示
- 工具执行默认禁用

当前不支持：

- 桌面前端
- 一键安装包
- 三端安装验证
- 真实工具执行
- 长期记忆写入
- 多 Provider 深度差异适配

## Mock 启动

```bash
python run_agent.py --mock --once "你好"
```

预期输出包含：

```text
[MOCK] 已收到：你好
```

进入交互：

```bash
python run_agent.py --mock
```

## 查看状态

```bash
python run_agent.py --mock --status
```

## 使用 OpenAI-compatible API

推荐使用环境变量，不建议把真实 API Key 写入仓库：

```bash
export TIANGONG_PROVIDER=openai_compatible
export TIANGONG_BASE_URL=https://api.example.com/v1
export TIANGONG_API_KEY=你的APIKey
export TIANGONG_MODEL=example-model
python run_agent.py --once "你好"
```

Windows PowerShell 示例：

```powershell
$env:TIANGONG_PROVIDER="openai_compatible"
$env:TIANGONG_BASE_URL="https://api.example.com/v1"
$env:TIANGONG_API_KEY="你的APIKey"
$env:TIANGONG_MODEL="example-model"
python run_agent.py --once "你好"
```

## 配置文件

示例配置位于：

```text
config/model_config.example.json
```

该文件不得写入真实 API Key。真实私有配置请另存到本地未入库路径。

## CLI 命令

交互模式中支持：

- `/help`：显示帮助
- `/status`：显示状态
- `/config`：显示脱敏配置
- `/reset`：重置当前会话
- `/exit` 或 `/quit`：退出

## 安全边界

- API Key 不应进入日志、报告或公共投影。
- 工具执行默认 `disabled`。
- 当前 CLI 只做最小对话闭环。
- 后续若接入工具，必须通过 RuntimeGovernedToolBridge 接 L6 治理链。
