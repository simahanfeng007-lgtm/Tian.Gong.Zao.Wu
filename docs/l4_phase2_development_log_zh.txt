四、核心语义
1. ActionPermitRef / ExecutionPermitRef 只是未来 L5 permit 引用，不是 L4 签发的许可。
2. PermitScope 只做结构范围表达，PermitExpiry 只识别显式 expiry 与 explicit_expired。
3. ActionGroundingGateValidator 只做结构校验，不做政策裁决或风险裁决。
4. allowed_for_grounding=True 只表示“结构上可进入动作落地承载”，不表示 L4 授权；结果对象保留 l4_authorized_action=False。
5. FakeBoundaryPermitForTestOnly 只能用于测试；production_path=True 时返回 TEST_ONLY_REJECTED。
6. L5 port 均为 Protocol 占位，不提供真实 L5 实现。
