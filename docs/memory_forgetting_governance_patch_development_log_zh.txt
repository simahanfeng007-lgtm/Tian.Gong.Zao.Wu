天工造物 记忆系统与遗忘系统专项补丁开发日志
日期：2026-06-04

本补丁按专项审阅报告补齐 L1-L4 的记忆/遗忘治理接口链，不实现真实记忆系统或遗忘系统。

完成内容：
1. L1 新增 memory_governance_ports.py，定义 MemoryGovernancePort 与 ForgettingGovernancePort。
2. L2 新增 memory_forgetting_state.py，定义保留、衰减、干扰、抑制、剪枝、修订、删除链、隐私边界等状态。
3. L2 MemoryRefState 增加 memory_kind 字段，保留 working / episodic / semantic / procedural / self / system 维度。
4. L3 新增 forgetting_service_request.py，定义遗忘治理请求、建议、状态转移建议、路由和外置化压力分数。
5. L4_L6_SURFACES 增加 memory_sink、forgetting_sink、retrieval_sink、learning_sink、affective_feedback_sink、privacy_retention_sink。
6. L4 新增 L4ToL6MemorySinkRequirement、L4ToL6ForgettingSinkRequirement 与 ActionResultCognitiveSinkHint。
7. 更新 L4→L5 / L4→L6 交接文档，明确记忆/遗忘 sink 链路。

边界：
1. 不读取真实记忆库。
2. 不写长期记忆。
3. 不执行召回、遗忘、删除、Tombstone、匿名化或隐私处理。
4. 不写 L2 状态。
5. 不实现 L5/L6。
