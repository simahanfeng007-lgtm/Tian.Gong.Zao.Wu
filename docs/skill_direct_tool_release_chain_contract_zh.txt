天工造物 L4 Skill 直显与工具释放链路契约

日期：2026-06-04

一、链路定位

本契约只补齐 Skill 直显、ToolGroup 释放、模型可见工具 schema、工具调用结果返回与 L3/L2 续接的显式引用链路。L4 仍只是动作落地承载层，不是执行智能体，不选择 Skill，不调用模型或工具，不签发许可。

二、字段映射

1. SkillProjection/Exposure
   - 输入引用：skill_projection_ref、skill_ref、trace_refs。
   - 输出到：SkillRequest/Selection。

2. SkillRequest/Selection
   - 输入引用：skill_request_ref、skill_selection_ref。
   - 输出到：SkillAuthorization/L5ReviewHint。

3. SkillAuthorization/L5ReviewHint
   - 输入引用：skill_authorization_ref、l5_review_hint_ref。
   - 输出到：ToolGroupRelease。
   - L4 不做授权，只携带未来 L5 审查提示。

4. ToolGroupRelease
   - 输入引用：tool_group_ref、release_ref、permit_ref。
   - 输出到：ReleasedToolSchemaView。

5. ReleasedToolSchemaView
   - 输入引用：tool_schema_refs、boundary_refs、permit_refs、audit_requirement_refs、evidence_refs、trace_refs。
   - 输出到：ModelVisibleReleasedToolView。
   - 不保存真实工具句柄。

6. ToolCallEnvelope
   - 输入引用：model_context_ref、visible_tool_schema_refs。
   - 输出到：ToolActionResult/ToolResultEnvelope。

7. ToolActionResult/ToolResultEnvelope
   - 输入引用：tool_call_ref、tool_action_result_ref、tool_result_envelope_ref。
   - 输出到：ModelContinuation/L3ReplanSuggestion。

8. ModelContinuation/L3ReplanSuggestion
   - 输入引用：model_continuation_ref、l3_replan_suggestion_ref。
   - 输出到：L2 StateUpdateSuggestion。

9. L2 StateUpdateSuggestion
   - 输入引用：l2_state_update_suggestion_ref。
   - L4 不写 L2，只返回建议引用。

三、硬边界

1. 不解析 SkillRegistry。
2. 不注册 ToolRegistry。
3. 不调用模型或工具。
4. 不签发 permit。
5. 不保存真实工具句柄。
6. 不恢复旧 Runtime、神枢或能力包主链。
