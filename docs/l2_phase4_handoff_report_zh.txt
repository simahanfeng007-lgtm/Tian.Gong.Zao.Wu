四、对象作用与边界

ControlPlaneState：
记录 run / task / Skill / ToolIntent / ActionIntent / ModelFeedback 关联的控制面状态、模式、边界、风险、资源、安全和策略引用；不执行控制、不应用模式、不路由任务。

ControlSignalState：
记录暂停、恢复、确认需求、降级等信号事实；不发送、不确认、不处理信号。

ControlConstraintState：
记录控制约束、适用对象、边界和信任边界引用；不实现拦截器或过滤器。

BoundaryCheckState：
记录某次边界检查状态、被检查对象、边界引用、风险视图引用、裁决引用、策略引用、证据和审计引用；不调用风险评分器、权限系统或确认票据系统。

BoundaryBlockedState / BoundaryDegradedState / BoundaryAlternativeState：
分别记录阻断、降级、替代路径事实；不执行恢复、不替换工具组、不改写计划、不自动选择替代路径。

RiskDecisionState：
记录风险视图、风险等级标签、外部裁决、策略、边界检查、证据和评分快照引用；不计算风险、不改变 severity、不做二次裁决。

PolicyReferenceState：
记录策略引用、短策略名、版本引用、适用对象和来源边界；不保存完整策略正文，不执行策略匹配。

DecisionRecordState：
记录外部裁决引用、来源、对象和过期引用；不实现 Decision 引擎。

ResourceBudgetState / QuotaState / RateLimitState / ResourceLeaseState / ResourcePressureState：
记录预算、配额、限速、租约和压力事实；不读取真实系统资源、不扣减预算、不 sleep、不重试、不创建或续租租约。

EnvironmentState / SandboxState / ExternalWorldReferenceState：
记录环境、沙箱和外部世界引用；不读取环境变量、不探测系统路径、不访问沙箱、不访问外部对象。

SecurityBoundaryState / PrivacyCredentialState / TrustBoundaryState / SecretReferenceState：
记录安全边界、隐私、凭据、密钥引用和信任边界；不执行安全扫描、不读取密钥、不验证身份。隐私与密钥引用状态默认并强制脱敏和缺值。
