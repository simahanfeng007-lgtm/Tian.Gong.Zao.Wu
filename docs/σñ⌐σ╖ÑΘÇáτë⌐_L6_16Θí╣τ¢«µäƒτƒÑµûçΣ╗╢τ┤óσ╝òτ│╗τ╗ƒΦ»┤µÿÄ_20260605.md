# 天工造物 L6.16 项目感知与文件索引系统

定位：给 LLM 装甲增加“项目雷达”，让模型在规划工程任务前获得安全的项目结构摘要。

## 新增能力

- `scan_project`：A1 只读工具，扫描工作区项目结构。
- `/scan [path]`：CLI 触发项目扫描。
- `/project`：显示最近项目索引。
- `/project-save <path>`：导出项目索引 JSON。
- `/project-reset`：清空当前进程内项目索引。
- ModelPlanner 自动接收项目结构摘要，辅助自然语言任务转计划。

## 安全边界

- 不修改 `tiangong_kernel/` 主体。
- 不读取完整源码内容，只采集路径、后缀、大小、角色等元数据。
- 不扫描 `.git`、`node_modules`、虚拟环境、缓存目录等高噪声目录。
- 跳过 `.env`、密钥、凭证、secret/token/password/credential 相关路径。
- 项目索引默认只保存在当前进程内，导出必须由用户显式触发。
- `scan_project` 只是 A1 只读感知工具，不开放写入、终端或网络能力。

## 推荐使用

```powershell
python run_agent.py --mock --tool-mode runtime_governed --planner-mode model_suggest --workspace ./workspace
```

交互模式：

```text
/scan .
/project
/plan 请检查项目，读取 README，然后打包交付
/run 请检查项目，读取 README，然后打包交付
```
