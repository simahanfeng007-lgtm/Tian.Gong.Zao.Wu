二、关键边界

1. Memory
- 最多 5 条 top_memory_hints。
- 只保留 sanitized_summary / recall_score / evidence_refs。
- 不注入原始记忆正文。
- 不写 memory。
- 不删 memory。

2. Affective
- 只输出 affective_style_hint 与 affective_candidate_bias。
- 七情只影响语言状态逻辑。
- 六欲只影响同风险等级内做事方式排序。
- 不授权。
- 不拒绝。
- 不调工具。
- 不调模型。
- 不改预算。
- 不覆盖质量门。

3. Lifecycle
- 最多 3 条 lifecycle_next_action_hints。
- 自愈、学习、自由意志、自迭代均为候选。
- 不抢当前用户任务。
- 不自动修复。
- 不自动合入。
- 不自动热切换。

4. Execution/P0
- 执行链 contract 永远优先。
- Provider 只给声明式约束，不裸调 SDK，不读明文凭证。
- Budget 只给低摩擦约束，不扣预算。
- Skill 只给 draft/hint，不激活。
- Handoff 只给父链回收提示，不自动递归派生。
- QualityGate 只阻断发布、打包、激活、注册、合入、热切换等边界。
