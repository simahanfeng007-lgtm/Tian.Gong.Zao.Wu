L6.72.44 任务流程显示开关接线表

一、目标
增加“显示任务流程”设置项，让用户可以在任务工作台界面与纯聊天界面之间切换。
该开关只影响显示层，不影响真实执行链。

二、设置项
名称：显示任务流程
默认：开启
持久化字段：show_task_flow
所在位置：设置页

三、开启时
- 首页显示任务工作台。
- 会话区显示 Codex 进度卡。
- 长链任务显示状态、百分比、进度条、Run 摘要、当前阶段、当前工具、步骤摘要。

四、关闭时
- 首页隐藏任务工作台。
- 会话区隐藏 Codex 进度卡。
- 保留纯聊天表面。
- stop / interrupt / reconnect 能力不删除；只是流程展示不露出。
- Runtime / Planner / ToolMode / QualityGate / Audit 事件链继续运行。

五、工作模式触发规则
- 聊天：正常聊天，不触发任务工作台。
- 干活：正常聊天表面，不自动触发任务工作台。
- 代码：正常聊天表面，不自动触发任务工作台。
- 文件：正常聊天表面，不自动触发任务工作台。
- 长链：才允许 planner_allowed/tools_requested/long_chain_requested 为 True，启动真实任务流程。

六、关键边界
- 前端不执行工具。
- 前端不裁决执行权限。
- 前端不替代 Runtime 状态。
- 关闭任务流程不等于关闭 Runtime。
- 开启任务流程不等于前端获得执行权。

七、smoke 覆盖
- chat/work/code/file 不触发任务流程。
- long_chain 触发任务流程。
- 普通聊天没有 Codex 进度卡。
- 长链任务生成 Codex 进度卡。
- 设置页存在“显示任务流程”。
- main_window_chat_runtime 中存在显示门控。
