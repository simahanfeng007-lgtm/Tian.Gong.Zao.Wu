L6.72.30 同页重复选择项精简表

| 页面/区域 | 原问题 | 修复后唯一入口 |
|---|---|---|
| 聊天页-工作模式 | 顶部栏 + 输入区双模式选择 | 仅聊天输入区可选工作模式 |
| 设置页-工作模式 | 设置页与顶部栏同时可改同一 work_mode | 设置页只读显示当前工作模式，不再编辑 |
| 状态栏-主题 | 状态栏永夜/极昼 + 设置页完整主题重复 | 仅设置页“字体与主题”可选主题 |
| 模型管理 | 残留旧模型设置函数可能重复渲染 | 仅 _build_model_management_card 作为模型管理入口 |
| 执行权边界 | 无 | 前端只提交模式/审批/配置，不直接执行工具 |
