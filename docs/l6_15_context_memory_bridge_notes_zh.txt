天工造物 L6.15 上下文/会话/记忆连续性桥说明

本阶段接入系统：上下文/会话/记忆连续性系统。

新增能力：
- RuntimeEntry 内置 ContextMemoryBridge。
- 每次 RuntimeRunResult 结束后生成安全摘要记录。
- /context 显示当前进程内上下文摘要。
- /context-save <path> 导出上下文摘要 JSON。
- /context-reset 清空上下文摘要。
- ModelPlanner 在 model_suggest/model_required 时获得最近安全摘要作为 planner hint。

安全边界：
- 不写长期记忆。
- 不保存完整 prompt。
- 不保存 API Key。
- 不保存完整文件内容。
- 不修改 tiangong_kernel 主体。
- 不新增真实工具权限。

价值：
- 降低长链断片概率。
- 让 LLM 计划器知道最近做过什么、产物在哪里、上一步是否失败。
- 为后续项目感知、工程修复、自修复、多智能体协作提供统一上下文入口。
