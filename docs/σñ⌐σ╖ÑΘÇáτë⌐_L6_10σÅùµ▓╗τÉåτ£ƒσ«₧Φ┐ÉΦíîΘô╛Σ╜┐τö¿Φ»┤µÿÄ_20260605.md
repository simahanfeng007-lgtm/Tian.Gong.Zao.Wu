# 天工造物 L6.10 受治理真实运行链使用说明

本阶段在 L6.9 外壳式启动器基础上新增 `tiangong_agent_runtime/`，用于打通最小真实执行链。

## 启动验证

```powershell
python run_agent.py --mock --once "你好"
```

## 受治理工具模式

```powershell
python run_agent.py --mock --tool-mode runtime_governed --workspace ./workspace --once "list ."
python run_agent.py --mock --tool-mode runtime_governed --workspace ./workspace --once "write demo.txt :: hello"
python run_agent.py --mock --tool-mode runtime_governed --workspace ./workspace --once "read demo.txt"
python run_agent.py --mock --tool-mode runtime_governed --workspace ./workspace --once "compileall ."
python run_agent.py --mock --tool-mode runtime_governed --workspace ./workspace --once "zip . dist/demo.zip"
```

## 当前开放的工具

- `list_dir`：A1，只读列目录。
- `read_file`：A1，只读文件。
- `write_workspace_file`：A3，工作区写入，覆盖前自动备份。
- `run_python_quality_check`：A3，只允许 `compileall` / `pytest`。
- `create_zip_package`：A3，工作区内打包 zip，并生成 SHA256。

## 执行策略

- A0/A1/A2/A3：自动执行。
- A4：生成确认票据，第一版不自动执行。
- A5：硬阻断。

## 关键限制

- 不执行任意 shell。
- 不写工作区外路径。
- 不访问 `.env`、密钥、凭证类文件。
- 不打包 secret/token/key/credential 类文件。
- 不接真实桌面控制。
- 不写长期记忆。
- 不修改 `tiangong_kernel/` 主体。

## 交互命令

进入交互模式后支持：

- `/tools`
- `/policy`
- `/workspace`
- `/audit`
- `/run <任务>`

示例：

```text
/run list .
/run write demo.txt :: hello
/run read demo.txt
```
